To-Do List App (Fixed Version)

This project is a simple To-Do List system written in **Python**.  
It started with faulty code provided as a class exercise, and our objectives was to fix the bugs, improve input validation, and practice collaboration using GitHub.


# What are the features? YOU CAN: #
- Add new tasks
- View all tasks
- Remove tasks by number (with validation)
- Input validation (no crashes on wrong input)
- Follows PEP8 coding style
